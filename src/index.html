☁ Sync across devices (GitHub)

Keeps this library identical on desktop, Chrome and Firefox by storing it in your GitHub repo (branch library-data). Needs a personal access token, entered once per device. The repo is public — synced lists are visible to anyone.

How to create the token (once, ~2 min)
  1. GitHub → your avatar → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
  2. Repository access: "Only select repositories" → roster-dataslate uniquement
  3. Permissions → Repository → Contents: Read and write (rien d'autre)
  4. Expiration : 1 an · Generate, puis colle le token ci-dessus sur chaque appareil

Le token reste dans le stockage local de cet appareil — il n'est jamais envoyé ailleurs que vers l'API GitHub, jamais synchronisé, jamais commité.
